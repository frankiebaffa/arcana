arcc --copy-file: Copy a file. All trailing arguments will be parsed as
                  --copy-file arguments until terminated by the '--' argument
                  or until all arguments have been consumed.

USAGE
  arcc -i -s <src> -d <dst> --

ARGUMENTS
  -s  --source       <src>  The path to the file.
  -d  --destination  <dst>  The path to which to copy <src>.

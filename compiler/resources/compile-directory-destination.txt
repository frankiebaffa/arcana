arcc --compile-directory --destination: Specification of how to output to the
                                        destination directory. All trailing
                                        arguments will be parsed as
                                        --destination arguments until terminated
                                        with the '--' argument or until all
                                        arguments have been consumed.

USAGE
  arcc... -d -d <dir> [-e <ext>] --

ARGUMENTS
  -d  --directory  <dir>  The directory to which to output the parsed templates.
  -e  --extension  <ext>  The file extension to use for the parsed templates.
                          The original file extension will be used if it not
                          included.

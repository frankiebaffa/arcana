arcc --compile-against-target: Compile against a specific alias in the
                               context(s). Can be included multiple times to
                               handle nested aliases.

USAGE
  arcc -a -r -h
  arcc -a -r -a <als> [-f] [-x] [-n] --...

ARGUMENTS
  -a  --alias               <als>  The alias to target.
  -f  --for-each                   Alias is an array and perform the compilation
                                   for each element.
  -t  --alias-to            <als>  The new alias for the target to be placed at
                                   from the root of the current context.
  -x  --filename-extractor  <arc>  Arcana instructions on how to extract the
                                   filename from the context object at the
                                   alias. Should be placed on the final target
                                   in the list.
